Description de la spécialité

Durant la formation, l’étudiant consolidera d’abord ses connaissances de base dans le domaine des logiciels et systèmes d’information répartis (architecture N-tier, JEE, etc.). Par la suite, il effectuera une montée en compétence dans les technologies suivantes : les technologies avancées du Web (Services Web, Réseaux Sociaux, Multimédia, etc.), les nouvelles architectures logicielles (SOA, Cloud), les Réseaux Sociaux, la programmation Mobile ou encore les processus métiers (BPM, ERP, etc).


Débouchés

La spécialité SIR vise des métiers de l’informatique à forte valeur ajoutée grâce aux compétences acquises dans trois domaines clés : les technologies avancées du Web, les nouveaux modes de développement et de déploiement de solutions logicielles et les processus métiers de l’entreprise.

Parmi les métiers visés on trouve :

  • Architecte en Systèmes d’Information,
  • Ingénieur Développement Logiciels,
  • Ingénieur Intégration,
  • Consultant ERP/BPM,
  • Chef de projet.

Finalité:   Professionnelle
